How I Got Here

I’ve always been amazed by the web and how quickly it's changed our daily lives forever. My first exposure to programming (beyond HTML) was when I was 13, leafing through a C++ book and being dumbfounded by the mathematical hieroglyphics I was looking at. I believed for a long time that it took a special kind of nutty genius to create the amazing programs and web apps I love using only a text editor. It seemed like magic without the illusion.

Fast-forward to the summer of 2014, I randomly came across a Reddit link to Harvard’s CS50 open course-ware that came with a lot of hype, so I gave it shot. Why not? After watching David Malan tear a phone book in half to demonstrate algorithms, and working through the Scratch problem set, the course immediately jumped into C. The hieroglyphics were back. I struggled through the first C problem set for many hours trying to make a Mario-style half pyramid as tall as the user desires. After a lot of frustration, and breaking the problem up into smaller and smaller pieces, the process started to make sense. I solved the first few problem sets and had more fun with it than I ever thought I would. By the start of 2015, I read about Free Code Camp.

More and more software had been moving to the web, and the idea of writing web apps that could run on any device with a JavaScript interpreter seemed really exciting. By that point, I was all in. I spent the next spring and the past summer learning HTML5, CSS3, JavaScript, and all the technologies that tie it all together. Nodejs, AngularJS, React, and all the server middleware and database options out there have been fundamentally changing the landscape of the web, and once Web Assembly becomes standardized, a whole new paradigm for the web will be possible. Fast, powerful applications that are currently not practical as web applications (think music and film production software) may finally reach the cloud in a few years. I’m working feverishly everyday to get to a level where I can help great companies make web apps no one believed were even possible just five years ago. The real work has barely begun.
